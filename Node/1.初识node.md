# 服务启动
## 什么是node？
   Node.js是一个基于chrome V8引擎的JavaScript运行时环境。它可以执行js代码。
## 开发框架
   Express快速、开放、极简的web开发框架
   Koa下一代web开发框架
   Egg为企业级框架和应用而生
## 启动node服务
   创建包文件 npm init
   安装依赖   npm i -s express body-parser
   启动 node xxx.js
   停止 CTRL + c
## 数据交互
   原生请求
